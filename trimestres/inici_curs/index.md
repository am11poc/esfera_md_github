# Tasques d'inici de curs

A continuació hi ha les tasques que cal fer a inici de curs per tal d'assegurar la millor gestió de les dades del centre:

* **[Supervisar els aspectes relacionats amb la matrícula](rev_mat/index.md)**

  + Definir els paràmetres per fer la matrícula [→](../../mgad1/configuracions/pc.md)
  + Dissenyar el currículum del centre [→](../../mgad1/curric/ccentre.md)

* **[Supervisar la fitxa de l'alumne/a](rev_fda/index.md)**

  + Actualitzar les dades dels alumnes: adreces, telèfons, … [→](../../mgad1/mat/fda/fda-ap/ccontactes.md)
  + Actualitzar les NESE, les mesures d'atenció a la diversitat i els suports dels alumnes [→](../../mgad1/mat/fda/fda-aa/atencio_d.md)
  + Actualitzar, si és el cas, la informació dels camps lliures [→](../../mgad1/mat/fda/fda-ap/camps_lliures.md)
  + Revisar les opcions de religió i llengües estrangeres en el currículum dels alumnes [→](../../mgad1/mat/fda/fda-aa/curriculars.md)
  + Actualitzar les dades relatives als serveis de menjador i transport, si és el cas [→](../../mgad1/mat/fda/fda-ap/serveis.md)

* **[Dissenyar i configurar els grups](grups/index.md)**

  + Revisar els grups assignats [→](../../mgad1/grups_i_doc/g_assignats.md)
  + Crear els grups classe i distribuir-hi els alumnes, els continguts, els professorts i el tutor/a. [→](../../mgad1/grups_i_doc/g_classe.md)
  + Crear, si escau, els grups complementaris [→](../../mgad1/grups_i_doc/g_complentari.md)
  + Crear, si escau, grups ZER [→](../../mgad1/grups_i_doc/g_zer.md)

* **[Preparar els documents](llistats/index.md)**

  + Elaborar i imprimir les **orles** [→](https://ateneu.xtec.cat/wikiform/wikiexport/esfera/mgap1/orles/index)
  + Elaborar i imprimir el **registre d’assistència** [→](https://ateneu.xtec.cat/wikiform/wikiexport/esfera/mgap1/rassistencia/index)
  + Elaborar i imprimir **llistats dels telèfons** [→](https://ateneu.xtec.cat/wikiform/wikiexport/esfera/mgap1/telefons/index)

* **[Personal](personal/index.md)**

  + Tramitar professorat pendent de nomenar [→](../../mgad1/personal/per_pn.md)
