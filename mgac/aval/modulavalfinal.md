## Avaluacions finals

Procés administratiu on d'acord al que ha establert la normativa es valoren els aprenentatges i les competències adquirides per l'alumne en finalitzar un ensenyament, nivell, mòdul,…

### Aplicació en els diferents ensenyaments

| Ensenyament | Av. final | Av. extraordinària | Observacions |
| --- | --- | --- | --- |
| **EINF** |  |  | No hi ha avaluacions finals |
| **EPRI** |  |  | A cada nivell, s'avaluen les àrees i les dimensions, a 2n, 4t i 6è s'avaluen també les competències transversals. |
| **ESO** |  |  | S'avaluen totes les matèries del currículum. |
| **BATX** |  |  | S'avaluen totes les matèries del currículum. |
| **CF d'FP** |  | | Es poden fer[1)](modulavalfinal.md#1) tantes avaluacions finals com sigui necessari. A cada sessió, s'avaluen totes les UF i Mòduls que l'alumne tingui a la matrícula i que no hagi superat ja en una sessió d'avaluació anterior. [2)](modulavalfinal.md#2) |
| **CF d'APD** |  | | Es poden fer fins a tres sessions avaluacions finals, la tercera extraordinària. A cada sessió, s'avaluen totes les UF i Mòduls que l'alumne tingui a la matrícula i que no hagi superat ja en una sessió d'avaluació anterior. [3)](modulavalfinal.md#3) |
